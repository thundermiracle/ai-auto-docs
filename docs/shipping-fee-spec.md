# 送料計算 仕様書

## 目的
注文金額に応じた送料を算出する。

## 用語
- 注文金額: 商品合計金額（円）。消費税・クーポン・ポイント等の扱いは本仕様の対象外。
- 送料: 本仕様で算出する配送料（円）。

## 料金ルール
注文金額に対して、以下のルールを上から順に適用する。

1. 注文金額が **4000円以下** の場合、送料は **1000円**。
2. 注文金額が **4000円超〜12999円以下** の場合、送料は **500円**。
3. 注文金額が **13000円以上** の場合、送料は **0円（送料無料）**。

## 境界条件
- 4000円はルール1に含める。
- 12999円はルール2に含める。
- 13000円はルール3に含める。

## 例
- 3000円 → 送料 1000円
- 4000円 → 送料 1000円
- 4001円 → 送料 500円
- 12999円 → 送料 500円
- 13000円 → 送料 0円
